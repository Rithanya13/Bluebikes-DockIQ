## Problem
BlueBikes stations frequently experience dock shortages or overflow during peak hours.
Expanding dock capacity is expensive and must be prioritized using data.

## Solution
DockIQ is a prescriptive analytics tool that recommends where and how many docks
should be added based on historical demand imbalance.

## Analytics Approach
1. Hourly arrivals and departures
2. Net flow imbalance
3. Dock Pressure Score
4. Actionable dock recommendations
